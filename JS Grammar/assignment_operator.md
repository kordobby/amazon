# 비교 연산자

## 대입 연산자

### 우측 피연산자의 값을 왼쪽 피연산자에 할ㅌㅈ당합

### 할당

```javascript
x = y; //  x = y
```

### 더하기 할당

```javascript
x += y; //  x = x + y
```

### 빼기 할당

```javascript
x -= y; //  x = x - y
```

### 곱하기 할당

```javascript
x *= y; //  x = x * y
```

### 나누기 할당

```javascript
x /= y; //  x = x / y
```

### 나머지 할당

```javascript
x %= y; //  x = x % y
```

### 거듭제곱 할당

```javascript
x **= y; //  x = x ** y
```

### 왼쪽 쉬프트 할당

```javascript
x <<= y; //  x = x << y
```

### 오른쪽 시프트 할당

```javascript
x >>= y; //  x = x >> y
```

### 부호 없는 오른쪽 시프트 할당

```javascript
x >>>= y; //  x >>> y
```

### 비트 AND 할당

```javascript
x & y; //  x = x & y
```

### 비트 XOR 할당

```javascript
x ^= y; //  x = x^y
```

### 비트 OR 할당

```javascript
x |= y; //  x = x | y
```

### 논리 AND 할당

```javascript
x && y; //  x && ( x = y )
```

### 널 병합 할당

```javascript
x ??= y; //  x ?? ( x = y )
```

### 할당

```javascript
x = y; //  x = y
```
