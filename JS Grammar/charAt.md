## charAt

### charAt 이란?

- 문자열에서 지정된 위치에 존재하는 문자를 찾아서 반환하는 함수

### charAt 사용 방법

```javascript
"[문자열]".charAt([문자위치]);
```

### charAt 예제

- 예제 #1
  - charAt 함수를 이용해 문자열 "charAt"의 3번째 위치 문자를 출력하시오.

```javascript
var str = "abcde";
str.charAt(2);
// str.charAt(2) : c
```
